Detailed Description

Loads a initial situation file.

This module loads an initial situation file.
The initial situation file contains a set of particules that are defined with their properties.
The file must have the format below :
Simple test with two particles 2 0.0,0.5,0.0004,0.0,0.5,0.01,4 0.5,0.5,0.0,0.0,0.5,0.01,3

The first line describes the fields and will be ignored
The second line contains the number of particles in the file (int)

The next lines describe the particles, one particle per line.
