\doxysubsection{Detailed Description}
Draw D\+I\+S\+Cs with S\+DL. 

This module exposes a few functions to ease the use of the S\+DL library, and a D\+I\+SC drawing routine.

Use -\/lm -\/l\+S\+DL when compiling in order to link with math library and S\+DL library.
